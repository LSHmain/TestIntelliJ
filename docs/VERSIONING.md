# LINKLE_Version

## 버전 가이드
### 주버전 (Major)
* 형식: X.0.0
* 호환성 깨지는 변화가 있을 때 올립니다.
* 예:
    * API나 데이터 구조 변경
    * 이전 버전과 호환되지 않는 기능 제거
    * 프로젝트 전면 리팩터링
* 예시: 1.5.2 → 2.0.0 (호환성 없는 큰 변화)

### 부버전 (Minor)
* 형식: 1.X.0
* 새로운 기능 추가 (기존 기능과 호환 유지)
* 예:
    * 새로운 API 추가
    * 기존 기능 확장
    * UI 개선
* 예시: 1.5.2 → 1.6.0 (기존 기능은 그대로, 새 기능 추가)

### 수정버전 (Patch)
* 형식: 1.0.X
* 버그 수정, 성능 개선, 테스트 기능, 사소한 변경
* 예:
    * 오타 수정
    * 로직 버그 수정
    * 보안 패치
* 예시: 1.0.2 → 1.0.3 (버그만 수정)
